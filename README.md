# Codex

A theme for those who work by candlelight — deep obsidian, burnished gold, and the gravity of a bound volume that has
survived centuries. Every surface evokes the cover of a codex: dark leather and gilt lettering, the hush of a library
after closing time. Champagne gold borders frame each element like the gilded edges of a folio page. The Literary
Universe pink appears as a ribbon marker — a single vivid thread through the darkness.

Designed for writers and creators who work best in the dark — focused, deliberate, and quietly authoritative.

## Color Philosophy

- **Background**: Deep charcoal / near-black — aged leather binding
- **Foreground**: Warm off-white — the illuminated page within
- **Primary accent**: LU pink (`#FF83D5`) — the silk ribbon marker
- **Secondary accent**: Champagne gold (`#D4A843`) — gilt lettering and edge gilding
- **Tertiary accent**: LU sky blue (`#7CC3EB`) — the blue ink of marginalia
- **Error / warning**: Vivid red — a wax seal broken

## Installation

```bash
omarchy-theme-install https://github.com/StorytellerCZ/Literary-Universe/omarchy-themes/lu-dark
```
